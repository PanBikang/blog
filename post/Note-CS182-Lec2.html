                              • Lecture 2

                              Reflex agent

                              Choose action based on current precept and do not consider the future consequences of their actions

                              planning agent

                              decision based on hypothesized consegences of actions

                              Most important, it consider how the world would be.

                              Search Problem

                              Search is a basic problem and model for planning agent.

                              A search problem consisits of:

                              • state space (s∈S)
                              • successor function (f(s,a)={s′})
                              • a start state and a goal test

                              A solution

                              is a sequence of action which transfroms the start state to a goal state

                              State Space Graphs

                              State space graph is a mathmatical representation of a search problem.

                              In a state space graph, each state occurs only once!

                              Search Trees

                              Nodes show states, but correspond to PLANS that achieve those states.

                              For most problems, we can never actually build the whole tree.

                              Each node in the search tree is an entire PATH in the state space graph

                              Search

                              • Expand out potential plans
                              • Maintain a fringe of potential plans under consideration
                              • Try to expand as few tree nodes as possible

                              Search Algorithm Properties

                              • Complete?
                              • Optimal?
                              • Time complexity?
                              • Space complexity?
                              • Suppose b is the branch factor and m is the maximum depth

                              Depth-First Search

                              Strategy: expand a deepest note first

                              Time complexity: O(bm)

                              Fringe space complexity: O(bm)

                              If m is infinite, it is not complete.

                              It is not optimal.

                              Breadth-First Search

                              Strategy: expand a shalloest node first

                              Suppose number of tiers have been searched is s

                              Time complexity: O(bs)

                              Fringe space complexity: O(bs)

                              It is complete.

                              It is optimal only if costs are all 1.

                              Iterative Deepening

                              Idea: get DFS's space advantage with BFS's time

                              Uniform Cost Search

                              Strategy: expand a cheapest node first

                              Suppose every arcs cost at least ε then the "depth" is roughly C∗ε

                              Time complexity: O(bC∗ε)

                              Fringe space complexity: O(bC∗ε)

                              It is complete and optimal

                              Shortage of Search and Models

                              However, all the search operates are all "in simulation"
