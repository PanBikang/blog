                                      Evaluation: Precision and Recall

                                      true positive: Predict positive and it is true.

                                      aaaa bbbbbbbb: predict bbbbbbbb and it is aaaa.

                                      Precision: % of items the system indentified as positive that are in fact positive.

                                      预测的样本里面有多少是准确的预测？

                                      Recall: % of positive items that were correctly identified by the system.

                                      真实上的样本数里面有多少正确的样本被召回了？

                                      Why don't we use accuracy as our metric?

                                      Imagine we save 1 million messages. 100 of them talked about "Pie" and 999900 talked abot something else. We could build a dumb classifier that just labels every message "not about pie", it would get 99.99% accuracy!!! In contrast, Recall = 0

                                      Evaluation: F-measure

                                      Fβ=(β2+1)PRβ2P+R

                                      Evaluation: Combine P/R

                                      Macro-averaging: calculate the prediction in a big table and compute the performance for each class, then average over classes.

                                      Micro-averaging: Aggregate statistics for all classes into one confusion matrix and compute the precision and recall from that table.

                                      Topic modeling

                                      Text clusters may correspond to different topics. So far, we assume a single cluster label for each document. But, a document may cover multiple topics.

                                      This is called probabilistic latent semantic analysis(pLSA).

                                      We can further add Dirichlet priors over topic & word distributions

                                      However, Dirichlet priors have some infinite probablistic value on the margin whenthe topic & word distribution is sparse. So we always need variational inference or MCMC to learn it.
