                  • Language Modeling

                  The effectiveness of RNNs is attributed to their ability to capture statistical contingencies that may span an arbitrary number of words.

                  LSTM

                  long short term memory

                  LSTM is a type of RNN as a solution to the vanishing gradients problem because the LSTM cell stores long-term information.

                  The LSTM can erase, write and read information from the cell. The selection of which information is erased/written/read is controlled by three corresponding gates.

                  The LSTM architecture makes it easier to preserve information over many timesteps

                  Notice that LSTM doesn't guarantee that there is no vanishing or exploding gradient, but it does provide an easier way to learn long-distance dependencies.

                  GRU

                  4 parts: Update gate, Reset gate, New hidden state content, Hidden state

                  Update gate controls what parts of the hidden state are updated/preserved.

                  Reset gate controls what parts of the previous hidden state are used to compute new content.

                   

                  Multi-layer RNN and Bidirectional RNN

                  Multi-layer RNN: Cascade the RNNs and sent the output of one RNN layer to another RNN layer.

                  Bidirectional RNN: connect two reverse direction RNN together. This structure cannot be use to language model.

                  Attention

                  The attention output mostly contains information from the hidden states that received high attention.

                  Dot-product Attention

                  Inputs: a query q and a set of key-value(k-v) pairs to an output.

                  Queries and keys have same dimension dk and values have dimension dv

                  Outpur: weighted sum of values weight of each value is computed by an inner product of query and corresponding key

                  A(q,K,V)=∑ieq⋅ki∑jeq⋅kjvi

                  We can stack multiple queries in a matrix Q

                  A(Q,K,V)=Softmax(QKT)V

                  Note that the Softmax here is a row-wise softmax.

                  Scaled Dot-product Attention

                  Problem: As dk gets large, the variance of qTk increases, which will lead to some very large values, and some numerical problem when applying exp in softmax.

                  A(Q,K,V)=Softmax(QKTdk)V

                  Attention Variants

                  There are several ways you can compute the attention score of a query vectore q and a key vector ki

                  • Basic dot-product attention
                  • Multiplicative attention (bilinear attention)
                  • Additive attention

                  Multi-head self attention

                  Problem with simple self-attention: only one way for words to interact with one-another.

                  Solution: First map Q,K,V into multiple lower dimensional spaces and apply attention in each space. Finally, concatenate outputs and pipe through a linear layer.
