introduction

In task-oriented communication, IB( information bottleneck) principle was used to formulize a rate-distortion tradeoff. What is so called rate-distortion is a tradeoff between the informativeness of encoded feature and the inference performance.

IB formulation aims at maximizing the mutual information between the inference result and the encoded feature and minimizing the mutual information between the encoded feature and the input data.

Then due to the intractable of IB formulation, we leverage the variational approximation to devise a tractable upper bound. This is VIB(virational information bottleneck). By selecting a sparsity -inducing distribution as the variational prior, the VIB framework prunes the redundant dimensions of the encoded feature vector to reduce the communication overhead. This is VFE(variational feature encoding).

The author also proposed a dynamic neural networks that can adaptively adjust the active dimensions according to different channel conditions.

The effectiveness is validated in both static and dynamic channel conditions on image classification tasks.

Building VFE

This paper try to prune the redundant dimensions in the encoded feature vector.

The author adopt the log-uniform distribution as the variational prior to induce sparsity, i.e.,

q(log⁡∣z^i∣)=constantq(\log|\hat{z}_i|) = \text{constant}q(log∣z^i​∣)=constant﻿.

The author then designing a dynamic neural network to control the encoded feature dimension flexibly.
